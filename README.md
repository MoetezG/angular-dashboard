# angular-dashboard

## Installation

1. Clone the repository:

```bash
  git clone https://github.com/MoetezG/angular-dashboard.git
  cd angular-dashboard
```

2. Install dependencies:

```bash
   npm  install
```

## Usage

### Run Server

```bash
run the server
cd server
npm run dev
```

### Run Server

```bash
run the client
cd client
ng serve
```
